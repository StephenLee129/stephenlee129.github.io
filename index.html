Projects
Jack of All Waters
Stephen Lee, Jeremy Fischer, Shiran Yuan, Nicholas Jean

project page / video

A water simulation created using Unity. Implemented features include realistic waves, buoyancy, and lighting effects, all running in real-time. I implemented the lighting effects using custom Unity shaders, which included reflections, refractions, caustics, and water fog.

Optimizing Ray-Marching using CUDA for Volumetric Rendering
Stephen Lee, Jeremy Fischer, Samuel Sovi

project page

Extension of a ray-tracer to handle participating media (translucent materials). Implemented a ray-marching algorithm to physically simulate lighting effects for these materials, and bounding volume hierarchies (BVH) to provide significant speedup. I primarily worked on optimizations including the work on BVH and memory management.

Simulating Wildfire Spread Using Physically Accurate Models
Stephen Lee, Anthony Mansur, Lindsay Smith

project page / code

Inspired by this work we created a wildfire simulation of our own using C++/CUDA to leverage parallelization on the GPU and OpenGL to render our scenes. I primarily worked on the physics-based fluid solver that enabled our physically accurate model by solving the Navier-Stokes equations.
